#   Modify the temperature conversion program fahr.c
#   using the following formats to format the ouput    

    The format are as follow:
        >> %d, %6d, %f, %6f, %.2, %6.2f.

    Each of these of these formats are used in each
    of the exercises in the current folder. So it 
    corresponds to the following relationship:

        For first.c ------->%d
        For second.c ----->%6d
        For third.c ------->%f
        For forth.c ------>%6f
        For fifth.c ----->%.2f
        For sixth.c----->%6.2f
